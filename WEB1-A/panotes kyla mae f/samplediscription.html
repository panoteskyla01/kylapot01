TPC
Transmission Control Protocol is a method (protocol) used along with the internet Protocol (IP) to send data in the form of message units, called packets, between computers over the Internet.
IP
Internet Protocol is the method or protocol by which data issent from one computer to another on the Internet. Each computer on the Internet is uniquely idenfied by an IP address.
FTP
File transfer Protocol is a protocol used to exchange files between computers on the Internet.
HTTP
Hypertext Transfer Protocol is the protocol used for exchanging text,sound, video, and other multimedia file on the Web.
